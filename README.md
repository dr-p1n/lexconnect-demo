# LexConnect — Demo Mockup

Interactive proposal mockup for the LexConnect lead — an ecosystem legal digital platform for Panama.

## Pages
- Homepage
- Search / lawyer directory
- Lawyer profile + booking flow
- Pricing (Free / Pro / AI Pro)
- Revista Jurídica (blog)

## Stack
Pure vanilla HTML/CSS/JS, single `index.html`. No build step. Deployed via Cloudflare Pages.

## Local preview
Just open `index.html` in a browser, or run `python3 -m http.server 8000`.
